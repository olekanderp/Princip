Максрос для автоматичного виконання дій

1. Всновлення необхідного пакету
sudo apt install xmacro


2. Команда після якої починається запис дій мишки і клавіатури. Після запуску команди, програма попросить натискнути 
кнопку після йокї запис команд зупиняється, коли натиснули кнопуку(зазвичай ESC клавіша) починається запис дій, по завершенню потрібно знову натискнути кнопку яку
нитискали при старті. 
Файл test.txt файл куди пишуться команди які виконував оператор(детальна інструкція за посиланням http://xmacro.sourceforge.net/

xmacrorec2 > test.txt


3. Запуск записаних команд із затримкою 120 мс

xmacroplay -d 120 < test.txt
